# Pousada Solar Dona Dora — Website v2

Site oficial da Pousada Solar Dona Dora em Capitólio, MG.

## Versão 2.0 — Mobile-first, Performático, Sem Animações Pesadas

### Características
- ✅ Mobile-first responsivo (testado em 360px+)
- ✅ Sem GSAP/Lenis — apenas CSS + Intersection Observer
- ✅ Logo como imagem
- ✅ Favicon configurado
- ✅ Página de Experiências
- ✅ Motor de reservas integrado (Foco Multimídia)
- ✅ SEO completo (robots.txt, sitemap.xml, schema.org)
- ✅ Touch targets ≥ 44px
- ✅ Hero com overlay escuro legível
- ✅ Performance otimizada (lazy loading, defer JS)

3. Settings → Pages → Source: main / root
4. Site disponível em `https://[usuario].github.io/[repo]/`
